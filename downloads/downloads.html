Downloads

Kusasa is a model of blended learning. It blends elearning with physical classroom activity. It also blends the reading of stories with the use of computers as an expressive medium. Which of these downloadable components will offer the greatest value to learners is an open question. If any of these materials can reignite the spark of creativity in learners and kindle the fire of reason then Kusasa will have achieved its objective of promoting analytical thinking.

Downloads
All Kusasa stories, LAMS modules, eToys projects and extras can be freely downloaded.

There are 24 stories, eight instructional stories, 24 LAMS modules, and 25 eToys tutorials and projects.

To run the eToys projects you will need to install the OLPC version of eToys: Squeakland-OLPC-080512.

To run the LAMS modules you will need to install LAMS which is available here.
